# Mistake - Edited generated block by hand

## What Happened

Hand-edited the generated block of the project map; the next rebuild silently discarded the edits.

## How To Avoid It

Generated markers mean generated content — write outside the markers.

## Prevention Task

Always run map rebuild instead of editing generated blocks; human notes go outside the markers.
